# Security Policy

## Reporting a Vulnerability

Do **not** open a public GitHub issue for security vulnerabilities.

Email: mahdi.awadi@gmail.com with subject `[gopkg security]`.

Please include:
- Affected package and version (e.g. `bus/v0.1.0`)
- A description of the vulnerability
- Steps to reproduce (if available)
- Suggested fix (if known)

You will receive acknowledgement within 72 hours. Coordinated disclosure timeline is typically 30 days after the fix ships, negotiable based on severity.

## Supported Versions

Pre-1.0 packages: only the latest minor version receives security fixes.

Post-1.0 packages: the latest major version receives fixes; previous major versions receive fixes for 6 months after a new major is tagged.
